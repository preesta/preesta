# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

The short version: assume good intent, criticize ideas not people, and don't be a jerk in issues, PRs, or discussions. Harassment, personal attacks, and discrimination are not welcome.

## Reporting

If someone is making the project space unpleasant, report it via GitHub's [private vulnerability reporting](https://github.com/preesta/preesta/security/advisories/new) (it's the same channel) or by email to the maintainer listed in `SECURITY.md`. Reports stay private; we'll deal with it without making a public scene.
